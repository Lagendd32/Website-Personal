INSTALASI DAN KONFIGURASI WEB SERVER

1.MENGENAL WEB SERVER

secara singkat, web server adalah perangkat lunak yang memberikan layanan atau service kepada client untuk menerima request HTTP maupun HTTPS dari client yang dikenal

client yang dimaksud adalah web browser (misalnya internet explorer,google chrome, dan mozilla firefox) setelah itu, web server akan mengirimkan request yang diminta client

dalam bentuk halaman web dan secara umum berbentuk dokumen HTML.

contoh web server , diantaranya Apache,Nginx,IIS(internet information services),dan lighttpd.

a.FUNGSI WEB SERVER

fungsi utama web server adalah memproses berkas-berkas yang diminta klien dan memberikan respon dalam bentuk halaman website yang berupa teks,gambar,vidio,dokumen, dan sebagainya.

proses ini dilakukan melalui protokol komunikasi yang telah ditentukan sedemikian rupa. oleh karena itu, ketika kalian mengakses suatu website pada saat itulah kalian melakukan

sebuah permintaan. jika alamat ip atau domain website benar,maka web server akan merespon permintaan tersebut dengan memberikan file atau data yang tersimpan pada file storage

atau database website tersebut

b.CARA KERJA WEB SERVER

telah diketahui bahwa semua protokol yang ada di internet melibatkan satu pihak sebagai server dan pihak lainnya sebagai klien. begitu pula dengan HTTP(hypertext transfer protokol)

yang merupakan wadah bagi aplikasi web yang dijalankan. dalam protokol HTTP, web server berperan sebagai pihak server.adapun web browser berperan sebagai pihak client. saat seorang

internet memasukkan sebuah alamat website melalui browser tertnetu(IE,Chrome,atau Firefox), browser akan memproses permintaan tersebut dengan mengirimkannya ke web server sesuai

alamat yang diminta. selanjutnya menunggu hasilnya untuk beberapa saat. apabila yang diminta oleh pengguna sebuah dokumen maka web server akan langsung mengirimkan dokumen tersebut

ke browser yang digunakan. tetapi pastinya bila dokumen yang diminta mengandung program server side tersebut dan mengirimkan hasilnya dalam bentuk HTML.
